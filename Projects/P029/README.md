## Reference Format
- Prefix: Uppercase characters (e.g., 'R', 'REQ')
- Number: 3 digits (e.g., '001', '042')
- Full reference: PREFIX + NUMBER (e.g., 'R001', 'REQ042')
- Usage: `- [ ] (R001) Requirement text`
